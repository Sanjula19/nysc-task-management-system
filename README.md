# NYSC Task Management System

A role-based task management system designed for the National Youth Services Council (NYSC).  
Built using PHP, MySQL, HTML, CSS, and JavaScript following clean architecture principles.

## Features
- Role-based access control (RBAC)
- Task creation and assignment
- Task status tracking
- Comment system
- Multi-user assignments
- Clean layered backend architecture

## Roles
- Chairman – Full control, creates and assigns tasks
- Director – Supervisory read-only access
- Deputy Director – Supervisory read-only access
- Assistant Director – Executes and updates tasks

## Tech Stack
- PHP (PDO)
- MySQL
- HTML/CSS/JavaScript
- Apache (XAMPP)

## Architecture
- Layered backend structure
- Service-based business logic
- Role-based authorization
- Many-to-many task assignments

## Status
Project in development — built step-by-step with architecture-first approach.
